# NomNom FTAP Roo Clone

This rebuilt clone is based on the fixed `Source/Strong/The Wourld` script.

## Usage

Use `NomNom.lua` as a standalone Roblox/FTAP client script in the same way as the original base script.

## Chat behavior

Automatic public-chat behavior has been removed. The rebuilt script does not auto-send startup messages through Roblox public chat APIs such as public Roblox chat services or public chat remotes.
