# Optional runtime modules

[`NomNom.lua`](../NomNom.lua:1) is still the supported standalone runtime. This [`modules`](README.md:1) folder is reserved for a future loader-compatible split if Xeno/runtime module loading becomes viable.

Current maintainable source/reference modules live in [`src`](../src:1); they document the bundled implementation and build strategy without changing runtime loading behavior.
